# 🎯 CSS Positions Demo

This project demonstrates different types of CSS positioning with simple HTML examples.

## 📂 Files Included

- Absolute.html
- Fixed.html
- relative.html
- static.html
- sticky.html

## 📌 CSS Position Types Explained

### 1. Static
- Default position
- Elements follow normal document flow

### 2. Relative
- Positioned relative to its normal position
- Can move using top, left, right, bottom

### 3. Absolute
- Positioned relative to nearest positioned ancestor
- Removed from normal flow

### 4. Fixed
- Fixed relative to viewport
- Does not move on scroll

### 5. Sticky
- Acts like relative until scroll
- Then becomes fixed
